<p><strong>We are pleased to announce the release of Merbridge 0.7.0!</strong></p>
<p>This release mainly includes two updates as below:</p>
<ol>
<li>
<p>Replaced XDP with <code>tc</code> (Traffic Control) to handle the ingress/egress traffic. This replacement comes from a situation that the XDP Generic mode has some problems and is not recommended to use in a production environment.</p>
</li>
<li>
<p>Merbridge now supports <a href="https://kuma.io/">Kuma</a>, a universal Envoy service mesh. Similar to Istio, you can use all capabilities provided by Merbridge when you are working in the Kuma environment. Thanks <a href="https://github.com/bartsmykla">@bartsmykla</a>, an excellent engineer from Kuma.</p>
</li>
</ol>
<p>For release notes, see <a href="https://github.com/merbridge/merbridge/releases/tag/0.7.0">Merbridge 0.7.0</a>.</p>
