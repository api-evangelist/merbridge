We are pleased to announce the release of Merbridge 0.7.0! This release mainly includes two updates as below: Replaced XDP with tc (Traffic Control) to handle the ingress/egress traffic. This replacement comes from a situation that the XDP Generic mode has some problems and is not recommended to use in a production environment.
